This Audio Recorder works in the latest Chrome Stable and Firefox Nightly

To run the recorder :

1 - Install python
2 - Download Audio_Recorder.zip, unzip it.
3 - In the shell, 'cd' into the "Audio_Recorder" directory.
4 - python -m SimpleHTTPServer.( Do not close the shell )
5 - Open Chrome and type "http://127.0.0.1:8000/index.html" in the address bar. 

The basic purpose is to run the "index.html" on a server.
